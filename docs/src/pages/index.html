Ionian is a fast and simple static site generator using Tera templates.

Getting started is as simple as creating a pages/index.html file:

<!DOCTYPE html>
<html lang="en">
    <head>
    	<meta charset="UTF-8">
    	<meta name="viewport" content="width=device-width, initial-scale=1.0">
    	<title>Hello</title>
    </head>
    <body>
        <h1>Hello World!</h1>
    </body>
</html>

And running ionian build to put a built site in `_build` — or ionian dev for a ridiculously fast dev server.

Extract the layout into layouts/base.html:

{% raw %}<!DOCTYPE html>
<html lang="en">
    <head>
    	<meta charset="UTF-8">
    	<meta name="viewport" content="width=device-width, initial-scale=1.0">
    	<title>{% block title %}{% endblock title %}</title>
    </head>
    <body>
        {% block content %}{% endblock content %}
    </body>
</html>
{% endraw %}
And simplify index.html: 
{% raw %}{% extends "layouts/base.html" %}
{% block title %}Hello{% endblock title %}
{% block content %}
<h1>Hello World!</h1>
{% endblock content %}
{% endraw %}

(For all Tera features, the root for templates is the root directory you specify.)

Add hooks to run external tools:

[[hooks]]
name = "tailwind"
during = "PreBuild"
dev = {Watch = ["tailwind.css"]}
command = "bun tailwindcss -i tailwind.css -o assets/app.css 2> /dev/null"
